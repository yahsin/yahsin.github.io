VisEngage Project

Engagement is an important aspect of user experience. While some researchers have investigated user engagement in the context of information visualization, there is still a lack of scholarship on the topic. After exploring the role and signiﬁcance of user engagement for information visualization, and discuss challenges in its characterization and assessment. We proposed VisEngage, a self-assessment questionnaire that provides insight into 11 different characteristics of user engagement.

How to use VisEngage?

The usage of this questionnaire is to be given to participants right after using the target visualization. The items describe participants’ experience or reaction, and ask for their degree of agreement. For each item, participants need to provide their response with 7 levels Likert-scale ranging from strongly agree to strongly disagree.

Example of VisEngage Questionnaire

Items

Aesthetics

  • A. While using this interactive chart, I found its look and feel to be pleasing.
  • B. The layout of this interactive chart is clear and balanced.

Captivation

  • C. While using this interactive chart, I felt absorbed to the extent that I was not aware of my surroundings.
  • D. While using this interactive chart, time seemed to pass quickly.

Challenge

  • E. While using this interactive chart, I enjoyed and accepted any challenges it presented.
  • F. While using this interactive chart, I had to think carefully, deeply, or reflectively.

Control

  • G. While using this interactive chart, its functions and features worked as I expected.
  • H. While using this interactive chart, I felt in control.

Discovery

  • I. While using this interactive chart, I learned something that I had not known before (e.g., a new fact, concept, or piece of information).
  • J. While using this interactive chart, I learned and figured out how to use it along the way.

Exploration

  • K. While using this interactive chart, I felt as though I was moving in or through it to learn about its content or message.
  • L. While using this interactive chart, I was exploring its features and content in a gradual fashion.

Creativity

  • M. While using this interactive chart, I found myself imagining things not directly related to what I was seeing in the chart.
  • N. While using this interactive chart, I found myself generating new and original thoughts or ideas.

Attention

  • O. While using this interactive chart, I found myself concentrating on specific aspects or features of the chart.
  • P. While using this interactive chart, I had to pay attention to multiple things at the same time.

Interest

  • Q. The content or message of this interactive chart was interesting to me.
  • R. The features or interactions provided in this interactive chart were interesting to me.

Novelty

  • S. The look and feel of this interactive chart was novel and fresh.
  • T. The features or interactions provided in this interactive chart were novel and fresh.

Autotelism

  • U. While using this interactive chart, I experienced enjoyment from the chart in and of itself, and not because it was a means to an end.
  • V. I would want to use this interactive chart if I saw it somewhere else and was not required or encouraged to use it.
