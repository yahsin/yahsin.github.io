VisEngage Project

Engagement is an important aspect of user experience. While some researchers have investigated user engagement in the context of information visualization, it has not received significant attention. We present an initial draft of a research instrument for assessing user engagement called VisEngage. VisEnage was developed by examining research in multiple disciplines and identifying phenomena relevant to user engagement in information visualization. Characteristics of these phenomena were identified and developed into a questionnaire that can be used to assess engagement.

How to use VisEngage?

The usage of this questionnaire is to be given to participants right after using the target visualization. The items describe participantsâ€™ experience or reaction, and ask for their degree of agreement. For each item, participants need to provide their response with 7 levels Likert-scale ranging from strongly agree to strongly disagree1.

IItems

Aesthetics

  • While using this interactive chart, I found its look and feel to be pleasing.
  • The layout of this interactive chart is clear and balanced.

Captivation

  • While using this interactive chart, I felt absorbed to the extent that I was not aware of my surroundings.
  • While using this interactive chart, time seemed to pass quickly.

Challenge

  • While using this interactive chart, I enjoyed and accepted any challenges it presented.
  • While using this interactive chart, I had to think carefully, deeply, or reflectively.

Control

  • While using this interactive chart, its functions and features worked as I expected.
  • While using this interactive chart, I felt in control.

Discovery

  • While using this interactive chart, I learned something that I had not known before (e.g., a new fact, concept, or piece of information).
  • While using this interactive chart, I learned and figured out how to use it along the way.

xploration

  • While using this interactive chart, I felt as though I was moving in or through it to learn about its content or message.
  • While using this interactive chart, I was exploring its features and content in a gradual fashion.

Creativity

  • While using this interactive chart, I found myself imagining things not directly related to what I was seeing in the chart.
  • While using this interactive chart, I found myself generating new and original thoughts or ideas.

Attention

  • While using this interactive chart, I found myself concentrating on specific aspects or features of the chart.
  • While using this interactive chart, I had to pay attention to multiple things at the same time.

Interest

  • The content or message of this interactive chart was interesting to me.
  • The features or interactions provided in this interactive chart were interesting to me.

Novelty

  • The look and feel of this interactive chart was novel and fresh.
  • The features or interactions provided in this interactive chart were novel and fresh.

Autotelism

  • While using this interactive chart, I experienced enjoyment from the chart in and of itself, and not because it was a means to an end.
  • I would want to use this interactive chart if I saw it somewhere else and was not required or encouraged to use it.
